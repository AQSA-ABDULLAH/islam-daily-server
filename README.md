# Welcome to islam-daily-Server

## To Get started

1. Install dependencies

   `npm install`

2. Start the app

   `npx run dev`

3. For mongodb View

- Add mongoDB Url (mongodb://localhost:27017/islam-daily).
- Click on connect button to establish database connection.

## OUTPUT

In the output, you'll find on consol:

- Server running on http://localhost:5000
- MongoDB connected
